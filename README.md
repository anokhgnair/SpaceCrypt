[![GitHub](https://img.shields.io/badge/GitHub-Repository-blue?logo=github)](https://github.com/anokhgnair/SpaceCrypt)

**SpaceCrypt** is a modern, user-friendly whitespace steganography tool built in Python. It lets you encode and decode secret messages using invisible whitespace characters (spaces, tabs, and newlines), hiding information in plain text. The app features a minimalist, intuitive GUI for easy encryption and decryption.

**Author:** Anokh G Nair

**Supported OS:** Windows


## Features ‚ú®

- **Encode & Decode Messages**: Hide and reveal messages using whitespace steganography.
- **Minimalist GUI**: Clean, simple, and intuitive interface.
- **Clipboard Support**: Copy encoded/decoded messages with one click.
- **Easy Navigation**: Switch between encoding and decoding from the main screen.
- **Input Guidance**: Input boxes provide clear instructions.
- **Error Handling**: User-friendly alerts for missing or invalid input.

## How to Use üìù

1. **Launch the App**: Run `spacecrypt.py`.
2. **Choose an Action**: Select `Encode` or `Decode` on the main screen.
3. **Encode**: Enter your secret message and click `Encode`. Copy the encoded whitespace text using the clipboard button.
4. **Decode**: Paste the encoded whitespace message and click `Decode`. The decoded message will be displayed for you to read or copy.
